## How to Get Started ðŸ’»

To set up qrinvent:

1. **Clone the Repo**:
   ```bash
   git clone https://github.com/arunpune/qrinvent
   cd qrinvent
   ```
2. **Install Dependencies**:
   ```bash
   npm install
   ```
3. **Setup your database:**

   - Create your supabase database: [Guide](./docs/get-started.md#development)
   - Run intial migrations

   ```
   npm run setup
   ```

4. **Start local server**:
   ```bash
   npm run dev
   ```
